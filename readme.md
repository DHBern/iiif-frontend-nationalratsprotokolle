## Installation

1. Install [Node.js](https://nodejs.org/en/https://nodejs.org/en/)
1. Install [yarn](https://yarnpkg.com) or [npm](https://www.npmjs.com/)
1. ```yarn install``` or ```npm install```
1. ```yarn run start``` or  ```npm run start```
